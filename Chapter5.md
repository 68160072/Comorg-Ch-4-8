## วัตถุประสงค์การเรียนรู้

หลังจากศึกษาบทนี้แล้ว ผู้อ่านควรจะสามารถ:

1. อธิบายภาพรวมของประเภทหลักของหน่วยความจำหลักแบบสารกึ่งตัวนำ (Semiconductor Main Memory) ได้  
2. เข้าใจการทำงานของรหัสพื้นฐานที่สามารถตรวจจับและแก้ไขข้อผิดพลาดแบบบิตเดียวในข้อมูลขนาด 8 บิตได้  
3. สรุปคุณลักษณะของโครงสร้าง DDR DRAM ที่ใช้งานในปัจจุบันได้  
4. เข้าใจความแตกต่างระหว่างหน่วยความจำแฟลชแบบ NOR และ NAND ได้  
5. อธิบายภาพรวมของเทคโนโลยีหน่วยความจำแบบโซลิดสเตตชนิดไม่ลบเลือน (Nonvolatile) รุ่นใหม่ ๆ ได้  

บทนี้เริ่มต้นด้วยการสำรวจระบบย่อยของหน่วยความจำหลักแบบสารกึ่งตัวนำ ซึ่งประกอบด้วยหน่วยความจำ ROM, DRAM และ SRAM จากนั้นจะกล่าวถึงเทคนิคการควบคุมข้อผิดพลาดที่ใช้เพื่อเพิ่มความน่าเชื่อถือของหน่วยความจำ และต่อด้วยการพิจารณาสถาปัตยกรรม DRAM ที่มีความก้าวหน้ามากขึ้น

## 5.1 หน่วยความจำหลักแบบสารกึ่งตัวนำ (Semiconductor Main Memory)

ในคอมพิวเตอร์ยุคแรก รูปแบบที่ใช้กันอย่างแพร่หลายของหน่วยเก็บข้อมูลแบบสุ่มเข้าถึง (Random-Access Storage) สำหรับหน่วยความจำหลัก คือการใช้อาร์เรย์ของวงแหวนแม่เหล็กรูปโดนัทที่เรียกว่า “คอร์” (Cores) ด้วยเหตุนี้ หน่วยความจำหลักจึงมักถูกเรียกว่า “คอร์”

อย่างไรก็ตาม การพัฒนาเทคโนโลยีไมโครอิเล็กทรอนิกส์ได้เข้ามาแทนที่หน่วยความจำแบบแกนแม่เหล็ก ปัจจุบันชิปสารกึ่งตัวนำถูกใช้อย่างแพร่หลายสำหรับหน่วยความจำหลักของคอมพิวเตอร์

### การจัดโครงสร้าง (Organization)

องค์ประกอบพื้นฐานของหน่วยความจำแบบสารกึ่งตัวนำคือ **เซลล์หน่วยความจำ (Memory Cell)** โดยมีคุณลักษณะสำคัญดังนี้

- มีสถานะเสถียรสองสถานะสำหรับแทนค่า 1 และ 0  
- สามารถเขียนข้อมูลเพื่อกำหนดสถานะของเซลล์ได้  
- สามารถอ่านข้อมูลเพื่อรับรู้สถานะที่เก็บอยู่ได้  

โดยทั่วไป เซลล์หน่วยความจำหนึ่งเซลล์จะมีขั้วต่อสามขั้ว ได้แก่

- **Select Terminal** – ใช้เลือกเซลล์หน่วยความจำสำหรับการอ่านหรือเขียน  
- **Control Terminal** – ใช้กำหนดว่าการทำงานเป็นการอ่านหรือเขียน  
- **Data Terminal** – ใช้รับหรือส่งข้อมูลของเซลล์  

ในกรณีของการเขียน ขั้วข้อมูลจะรับสัญญาณเพื่อกำหนดค่า 1 หรือ 0  
ในกรณีของการอ่าน ขั้วข้อมูลจะส่งสถานะของเซลล์ออกมาเป็นสัญญาณเอาต์พุต

รายละเอียดเชิงลึกเกี่ยวกับโครงสร้างภายใน การทำงาน และจังหวะเวลาของเซลล์หน่วยความจำขึ้นอยู่กับเทคโนโลยีวงจรรวมที่ใช้ ซึ่งอยู่นอกขอบเขตของบทนี้ โดยในที่นี้จะถือเป็นสมมติฐานพื้นฐานว่าเซลล์แต่ละเซลล์สามารถถูกเลือกเพื่อดำเนินการอ่านหรือเขียนข้อมูลได้

## รูปที่ 5.1 การทำงานของเซลล์หน่วยความจำ

รูปที่ 5.1 แสดงการทำงานของเซลล์หน่วยความจำ โดยแบ่งเป็นสองกรณี

- (a) การเขียนข้อมูลเข้า (Write Operation)  
  ใช้สัญญาณ **Control, Select และ Data In**

- (b) การอ่านข้อมูลออก (Read Operation)  
  ใช้สัญญาณ **Control, Select และ Sense**

## DRAM และ SRAM

หน่วยความจำทุกประเภทในบทนี้เป็นหน่วยความจำแบบ **Random Access**  
กล่าวคือ ข้อมูลแต่ละคำ (word) สามารถถูกเข้าถึงได้โดยตรงผ่านวงจร **wired-in addressing logic**

ตารางที่ 5.1 แสดงประเภทหลักของ **Semiconductor Memory**

แม้ว่าจะเรียกว่า **Random-Access Memory (RAM)** แต่จริง ๆ แล้วหน่วยความจำทุกชนิดในตารางสามารถเข้าถึงแบบสุ่มได้ทั้งสิ้น

คุณสมบัติสำคัญของ RAM

- สามารถ **อ่านและเขียนข้อมูลได้**
- ใช้ **สัญญาณไฟฟ้า** ในการอ่านและเขียน
- เป็นหน่วยความจำแบบ **Volatile** (ข้อมูลหายเมื่อไฟดับ)

RAM แบบดั้งเดิมมีสองชนิดหลัก

- **DRAM (Dynamic RAM)**
- **SRAM (Static RAM)**

RAM รุ่นใหม่บางประเภท (กล่าวถึงในหัวข้อ 5.5) เป็นแบบ **Nonvolatile**

---

## Dynamic RAM (DRAM)

DRAM เป็นหน่วยความจำที่เก็บข้อมูลในรูปของ **ประจุไฟฟ้าบนตัวเก็บประจุ (Capacitor)**

- มีประจุ = บิต **1**
- ไม่มีประจุ = บิต **0**

เนื่องจากตัวเก็บประจุมีการรั่วของประจุไฟฟ้า จึงต้องมีการ

**Refresh Charge (รีเฟรชประจุ)** เป็นระยะ  
เพื่อรักษาข้อมูลที่เก็บไว้

## ตารางที่ 5.1 ประเภทของหน่วยความจำแบบสารกึ่งตัวนำ

| Memory Type | Category | Erasure | Write Mechanism | Volatility |
|---|---|---|---|---|
| RAM | Read-Write Memory | Electrically (byte-level) | Electrically | Volatile |
| ROM | Read-Only Memory | Not possible | Masks | Nonvolatile |
| PROM | Read-Only Memory | Not possible | Electrically | Nonvolatile |
| EPROM | Read-Mostly Memory | UV light (chip-level) | Electrically | Nonvolatile |
| EEPROM | Read-Mostly Memory | Electrically (byte-level) | Electrically | Nonvolatile |
| Flash Memory | Read-Mostly Memory | Electrically (block-level) | Electrically | Nonvolatile |
